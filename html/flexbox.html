Flexbox

Trabalhando com flexbox

O flexbox trabalha com 2 eixos, o eixo principal (na horizontal); e o eixo transversal (na vertical). O comportamento dos eixos é definido pela propriedade CSS flex-direction, é ela quem vai ditar as regras dos eixos. Possui 4 valores principais:
- row
- row-reverse
- column
- column-reverse

Align-items

Podemos pensar nessa propriedade como se fosse o Justify-content mas para alinhar os itens no eixo transversal, ou seja, na vertical.
Também é uma propriedade de flex-container.

Flex Wrap:

O flex wrap define se os itens devem ou não quebrar a linha, e move os itens ao longo da linha transversal. por padrão não a quebra.
Também é uma propriedade de flex-container.
