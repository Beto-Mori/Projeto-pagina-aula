CSS inline, interno e externo. Qual o melhor?

CSS inline - NÃO RECOMENDADO :

Esta abordagem não é recomendada, pois a manutenção do código se torna desnecessariamente trabalhosa e repetitiva

CSS interno - NÃO RECOMENDADO :

Para esta estilização ser aplicada: colocariamos uma tag style no head da página e a estilização seria feita de forma parecida com o CSS padrão, seria mais fácil a manutenção porém ainda não é o ideal.

CSS externo - RECOMENDADO :

Esta abordagem é a recomendada para a estilização de páginas HTML, pois deivide as funções para seus respectivos diretórios.
